# Report-First Static Previews

These previews are non-production static authoring artefacts.

They are not runtime result pages, do not alter scoring, and do not change persisted results. They exist to test reader experience, report structure, visual hierarchy, and future payload/rendering feasibility for the report-first proof track.

Generated HTML previews should exclude `Editorial QA Notes` from the reader-facing report body. Any authoring/debug appendix is for local review only and must not be treated as production result rendering.
