# 06 Flow Recognition Accessibility Review Prompt

Use this prompt after the language is near final.

## Task

Review the full result journey for a selected Flow State pattern.

Check:

- recognisability
- plainness
- reading flow
- mobile density
- application usefulness
- whether the result feels like a behavioural intelligence briefing

## Output

Return:

- overall judgement
- strongest sections
- weakest sections
- confusing phrases
- density problems
- practical usefulness gaps
- top five upgrade recommendations

Do not rewrite the whole result. Recommend targeted changes only.
