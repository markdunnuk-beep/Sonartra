# Legacy Archive Note

The WPLP files, scripts, seed data, and supporting docs are archived reference material only.

They must not drive active implementation for the single-domain ranked-pattern engine.

## Superseded Requirements

The old WPLP and multi-domain acceptance tests are no longer valid active product requirements.

Superseded items include:

- WPLP-80 canonical seed strategy
- WPLP workbook conversion
- WPLP seed verification as active release proof
- multi-domain assessment authoring
- multi-domain result payload expectations
- question-section and signal-group domain modelling
- overlay dimensions
- thresholds, archetypes, sentence libraries, and old rule-engine rows

## Superseded Single-Domain Model

The old pair-oriented single-domain language model is superseded by the ranked-pattern package.

The active model is:

- one domain
- exactly four scored signals
- twenty-four ranked signal patterns
- four score shapes
- deterministic option-weight scoring
- persisted canonical result payload with reader-first sections from the import package

Legacy files may remain in the repository until a dedicated cleanup task removes or archives them more formally. During that transition, they must not be treated as current source of truth.
